Bloqra is a modern, professional WordPress block theme and the free companion to the Bloqra Gutenberg blocks plugin.

== Description ==

Bloqra is a fast, modern block theme (full site editing) that looks polished the moment you activate it. It ships with a fully designed homepage — hero, feature grid, highlight, blog and call-to-action sections — plus ready-made About and Contact page patterns and styled WooCommerce support.

Everything is driven by theme.json, so you can restyle the whole site (colors, typography, spacing) in seconds from the Site Editor. Bloqra is the free companion theme for the Bloqra Gutenberg blocks plugin (https://wordpress.org/plugins/bloqra/), but it works beautifully on its own with core blocks alone.

Features:

* Full Site Editing with the WordPress Site Editor
* A designed homepage out of the box (previewable without setup)
* Curated block patterns: header, footer, hero, features, highlight, blog, CTA, About and Contact
* Vibrant, professional violet color palette with a complete tonal scale
* Fluid typography and spacing driven by theme.json
* Custom block styles: Subheading, Card, Checklist, Pill and Gradient separator
* WooCommerce support, styled to match the theme
* Translation ready, accessible focus styles and clean, lightweight markup
* No page builder and no required plugins

== Installation ==

1. In your WordPress dashboard, go to Appearance > Themes > Add New.
2. Search for "Bloqra", then click Install and Activate.
3. Open Appearance > Editor to customize templates, styles and patterns.
4. (Optional) Install the free Bloqra blocks plugin from
   https://wordpress.org/plugins/bloqra/ for even more building blocks.

== Frequently Asked Questions ==

= Do I need the Bloqra plugin to use this theme? =

No. Bloqra is a standalone block theme built with core WordPress blocks. The
Bloqra blocks plugin is optional and simply adds more patterns and blocks.

= How do I edit the homepage? =

Go to Appearance > Editor > Templates > Front Page. Every section is a block
pattern you can edit, reorder or remove.

= Does it work with WooCommerce? =

Yes. When WooCommerce is active, Bloqra loads matching styles for the shop,
product, cart, checkout and account pages.
